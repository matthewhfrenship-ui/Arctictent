Lightweight demo using Three.js. Open the demo in a browser by serving this folder as static files.

Quick preview (from the workspace root):

```bash
# start a small HTTP server on port 8000
python3 -m http.server 8000

# then open in your browser:
http://localhost:8000/index.html
```

Controls (brief):
- Arrow keys: orbit camera (hold Shift then arrows to rotate player)
- W/A/S/D: move
- Q: dash
- X: attack, F: block (hold), R: respawn
- O/I or +/-: zoom
- M: open in-game menu (Save/Load/Delete)

File: `index.html`
